# Visual Admin Development Pattern (DDG Buro)

Этот микроскилл определяет основной паттерн разработки компонентов и эффектов на проекте NEW_Denis-Daragan-Garden-Syte-2026. 

Вместо хардкода значений в CSS или сложных админ-панелей с бэкендом, мы используем паттерн "Визуального билдера с localStorage".

## Шаги для реализации концепции:

1. **Создание скрытого роута редактирования:**
   Для каждой визуально сложной страницы или компонента (например, `/info`) создавайте внутренний роут для редактирования (например, `/info/edit`).

2. **Сборка визуального пульта управления (Control Panel):**
   На странице редактирования внедряйте удобный UI для настройки: ползунки (`<input type="range">`), кнопки переключения состояний, визуальные текстовые редакторы (`contentEditable`).

3. **Связывание с localStorage (State Persistence):**
   Любые изменения в пульте управления должны использовать React `useState` и `useEffect` для мгновенного сохранения состояния (как текста, так и числовых параметров настройки) в `localStorage` браузера.

4. **Инъекция CSS-переменных (Procedural Effects):**
   Настройки из пульта управления (яркость, интенсивность грязи, шум, контраст) передаются в компоненты через `style={{ '--dynamic-var': value }}`. В CSS обращайтесь к этим переменным через `var(--dynamic-var)`.

5. **Публичный рендер (Public Render):**
   На публичной странице (которую видят пользователи, например `/info`):
   - Считывайте те же самые ключи из `localStorage` при монтировании (`useEffect`).
   - Рендерите структуру DOM точно так же, как и в редакторе (те же CSS классы и переменные).
   - ПОЛНОСТЬЮ убирайте пульт управления (Control Panel).
   - Выключайте возможность редактирования (`contentEditable={false}`, `pointer-events: none` для текста).

## Преимущества подхода:
- **1:1 WYSIWYG:** Клиент видит именно то, что настраивает.
- **Micro-Admin:** Не требуется бэкенд или база данных для хранения настроек фронтенда.
- **Дизайнерская свобода:** Клиент может сам искать идеальный баланс пикселей, шума и эффектов без помощи программиста.
