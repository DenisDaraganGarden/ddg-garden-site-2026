# Portfolio Images Directory

This is where you drop images for your 3D vinyl records.

In `src/components/3d/VinylGallery.jsx`, you can link your data items to these textures.
For example, create folders like:

- `public/portfolio/project-alpha/cover.jpg`
- `public/portfolio/brutalist-garden/cover.png`

And in the code you will pass `imageUrl="/portfolio/project-alpha/cover.jpg"` to the 3D objects.
